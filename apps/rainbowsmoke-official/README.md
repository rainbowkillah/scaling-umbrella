# rainbowsmoke-official

This is a Cloudflare Worker application for the RainbowsmokeOfficial account, generated and managed with [Nx](https://nx.dev).

## Development

Run the worker locally:
```bash
nx dev rainbowsmoke-official
```

## Deployment

Deploy to Cloudflare:
```bash
nx deploy rainbowsmoke-official
```

Make sure to configure your `account_id` in `wrangler.toml` before deploying.
