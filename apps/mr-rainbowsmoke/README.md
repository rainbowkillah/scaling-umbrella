# mr-rainbowsmoke

This is a Cloudflare Worker application for the Mr. Rainbowsmoke account, generated and managed with [Nx](https://nx.dev).

## Development

Run the worker locally:
```bash
nx dev mr-rainbowsmoke
```

## Deployment

Deploy to Cloudflare:
```bash
nx deploy mr-rainbowsmoke
```

Make sure to configure your `account_id` in `wrangler.toml` before deploying.
